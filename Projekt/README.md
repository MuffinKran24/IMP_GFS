# Ein einfaches Thermometer, das die Temperatur und Luftfeuchtigkeit auf einer Flüssigkristallanzeige anzeigt

## Bauteile:
- Arduino UNO R3
- LCD-1602A
- DHT11 Humidity & Temperature Sensor
- 10K Potenziometer
- eine Menge Kabel

## Abhängigkeiten:
- Arduino IDE
- DHT sensor library
- LiquidCrystal library
